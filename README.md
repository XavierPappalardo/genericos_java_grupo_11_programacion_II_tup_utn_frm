Ejercicio 3: Par de números e Inversión con Genéricos

Usando clases genéricas en java, podemos ingresar 2 números, invertirlos y asegurar su integridad.



Funcionalidades del Código:

* Creación de una clase genérica con varios atributos
* Uso de parámetros de tipo <A, B>
* Método para mostrar e invertir los pares ingresados
* El usuario puede ingresar los pares de números por teclado a la lista.





Ejercicio 4: Calculadora con Genéricos

Usando clases genéricas en java, podemos trabajar con listas de distintos tipos de datos sin duplicar código.



Desarrollamos una estructura genérica que puede almacenar elementos de diferentes tipos y realizar operaciones básicas



Funcionalidades del Código:

\- Creación de una clase genérica que maneja listas.

\- Uso de parámetros de tipo (<T>)

\- Inserción de elementos en la lista

\- Mostrar los elementos almacenados

\- El usuario puede ingresar valores por teclado a la lista
